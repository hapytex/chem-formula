Conversion to canonical forms

toMolecular :: (Eq a, Hashable a) => Formula a -> Formula a Source #

Convert the given Formula to the a Formula that is a molecular formula. In this formula, each atom is denoted once with the total number of times it occurs.

toHillFormula :: (Eq a, Hashable a, HillCompare a) => Formula a -> Formula a Source #

Convert the given Formula to a Formula that presents the molecule with the Hill system. In this system carbon is denoted first followed by hydrogen, etc.
